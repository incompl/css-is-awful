Awful Thing #2

Clearfix

The Problem

I'm floating and I'm poking out of my container!

Old Clearfix

          .clearfix:after {
               visibility: hidden;
               display: block;
               font-size: 0;
               content: " ";
               clear: both;
               height: 0;
               }
          .clearfix { display: inline-block; }
          * html .clearfix { height: 1%; }
          .clearfix { display: block; }
I'm fixed!

New Clearfix

          .clearfix {
            overflow: auto;
          }
I'm fixed!

IE8+

Clearfix Everything

          * {
            overflow: auto;
          }
Everything is fixed!
